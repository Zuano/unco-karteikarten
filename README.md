# UNCO Karteikarten — 2. SBO 30.04.2026

Interaktive Buchhaltungs-Karteikarten zur 2. Schriftlichen Überprüfung in Unternehmensrechnungswesen (2AHB), Kontenplan 2. Semester.

**Themen (40 Karten gesamt):**
- Rücksendungen
- Rechnungsausgleich, Mahnspesen, Verzugszinsen
- Kauf Anlagengüter inkl. GWG
- Bezugs- und Versandkosten

Pro Thema gibt es zwei Übungen mit je 5 Karten:
- ✏️ **Selbst ausfüllen** — Konto-Nr. und Betrag eintippen
- 📋 **Multiple Choice** — Konto aus Dropdown wählen

Bei richtiger Antwort: grünes Feedback + Erklärung.
Bei falscher Antwort: orange **Lösungs-Karteikarte** klappt darunter auf.

## Datenschutz

Keine Server-Anbindung, kein localStorage, keine Cookies, kein Login. Punktestand und Antworten leben nur im Browser-RAM und sind beim Reload weg. Jede:r Studierende löst die Übung völlig getrennt.

## Selbst aufsetzen (GitHub Pages, ~30 Sekunden)

1. https://github.com/new → Repository-Name z. B. `unco-karteikarten`, **Public**, „Create repository".
2. Auf der neuen Repo-Seite: „uploading an existing file" → `index.html` per Drag & Drop reinziehen → „Commit changes".
3. **Settings → Pages → Source: „Deploy from a branch", Branch: `main` / Folder: `/ (root)` → Save.**
4. Nach 1–2 Minuten ist die Seite unter `https://<dein-username>.github.io/unco-karteikarten/` live.

## Alternativ ohne GitHub

- [Netlify Drop](https://app.netlify.com/drop) — einfach `index.html` aufs Browserfenster ziehen, fertig.
- [Cloudflare Pages](https://pages.cloudflare.com/) — etwas mehr Setup, kostenlos.
